# lain

A Wikipedia interface styled after Serial Experiments Lain's cyberpunk aesthetic.

## Features

- **Retro-tech aesthetic**: Dark green terminal-style interface with CRT effects
- **Wikipedia search integration**: Real-time search using Wikipedia's API
- **Cyberpunk design**: Scanlines, glitch effects, and phosphor-green color scheme
- **Responsive layout**: Works on desktop and mobile devices

## Usage

Simply open `index.html` in a web browser to access the interface.

1. Enter your search query in the search box
2. Press Enter or click the SEARCH button
3. Browse Wikipedia articles in the signature Lain style

## Technical Details

- Pure HTML/CSS/JavaScript - no dependencies required
- Uses Wikipedia's public API
- Client-side only - can be hosted anywhere
- Features include:
  - CRT scanline effects
  - Glitch animations
  - Phosphor green terminal styling
  - Hover effects and transitions

## Theme

Inspired by Serial Experiments Lain, this interface captures the show's aesthetic:
- Monospace fonts
- Dark terminal backgrounds
- Green phosphor text (#00ff00)
- Cyberpunk visual effects
- "Present day, present time" references
